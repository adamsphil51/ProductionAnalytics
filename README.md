# Dawa_ProductionAnalytics

This project demonstrates **drug manufacturing and production analytics** for a pharmaceutical manufacturer in Kenya.  

**Objectives:**  
- Track **production batches** and inventory levels  
- Analyze **drug output per batch**  
- Monitor **distribution of manufactured drugs** to hospitals and pharmacies  
- Identify **production bottlenecks** and optimize supply chain  

**Skills demonstrated:** SQL, Production Analytics, Inventory Management, Supply Chain Analytics, Business Intelligence

**Drugs included:** Paracetamol 500mg, Amoxicillin 250mg, Ibuprofen 400mg
